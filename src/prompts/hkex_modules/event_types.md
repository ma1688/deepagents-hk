# 📚 HKEX 财技事件类型定义

## 三大类型对比

| 维度 | 🔵 供股 | 🟢 配售 | 🔴 全购/要约 |
|------|---------|---------|-------------|
| **英文** | Rights Issue | Placement | General Offer |
| **定义** | 向现有股东按比例发行新股 | 向特定投资者配售股份 | 收购方向全体股东提出收购要约 |
| **对象** | 所有现有股东 | 选定的机构/个人 | 全体股东 |
| **选择权** | 可认购/放弃/出售权益 | 无优先权 | 可接受/拒绝要约 |
| **触发条件** | 公司需集资 | 快速集资需求 | 持股达30%触发强制要约 |
| **结果公告** | 《供股結果》 | 《配售完成》 | 《要約結果》 |

## 快速判断规则

```
公告标题/内容包含 → 判定类型

供股、供股權、認購權、按比例、每持有 → 🔵 供股
配售、授權配售、先舊後新、配售協議 → 🟢 配售
要約、全購、強制性、收購要約、無條件 → 🔴 全购
```

# 关键词速查表

> 💡 HKEX公告使用**繁体中文**，以下同时列出繁/简体

## 🔵 供股关键词

### 阶段性关键词

| 阶段 | 关键词 (繁/简) | 示例标题 |
|------|---------------|---------|
| 建议 | `建議供股` / `建议供股`、`擬進行` / `拟进行` | 《建議供股》 |
| 比例 | `按比例`、`每持有X股`、`供股基準` / `供股基准` | 《每持有2股可認購1股》 |
| 发售 | `公開發售` / `公开发售`、`公開招股` | 《公開發售新股》 |
| 文件 | `供股章程`、`暫定配額通知書` / `暂定配额通知书` | 《寄發供股章程》 |
| 权益 | `認購權` / `认购权`、`供股權` / `供股权`、`額外申請` / `额外申请` | 《有關認購權之公告》 |
| 结果 | `供股結果` / `供股结果`、`認購結果` / `认购结果` | 《供股結果》 |

### 正则表达式
```regex
(供股|供股權|認購權|按比例|每持有)
```

## ⚠️ 排除关键词

以下标题通常与财技事件无关，应**排除**：

- `證券變動月報表` / `证券变动月报表` — 月度例行披露
- `翌日披露報表` / `翌日披露报表` — 期权/奖励股份
- `董事名單` / `董事名单` — 人事变动
- `更改公司名稱` / `更改公司名称` — 名称变更
- `股息` — 纯派息公告
- `月報表` / `月报表` — 例行披露
